## Overview

CRM Analytics Academy turns scattered SAQL knowledge into a structured, searchable
curriculum. It's built on this very theme — the **Swarnil CSS** framework — so every
lesson gets fast, accessible, dark-mode-ready typography for free.

## What I built

- A docs information architecture with category landing pages and a sticky table of contents.
- Reusable lesson components: callouts, steppers, and copy-ready code blocks.
- An AdSense layout that never causes layout shift, even on slow connections.

> Engineering meets storytelling — the goal was a site that *teaches* as cleanly as it *reads*.
